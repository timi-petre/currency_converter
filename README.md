# currency_converter

Currency Converter

Author : @timi-petre 

Aplicatia are in componenta sa:

    - Imagine
    - TextField
    - ElevatedButton
    - Text
  
  TextField contine:

    - prefix
    - text
    - suffix


Cand prefixul este apasat, se va sterge infrmatia din TextField, si din Text de sub ElevatedButton.
